Security practices

Security

Worklog Rollup for Jira is designed as an Atlassian Forge app for Jira Cloud.

Platform

Atlassian Forge and Jira Cloud.

Scopes

read:jira-work and storage:app.

Data read

Issue hierarchy and Jira time tracking fields needed for calculation.

Cache

Short-lived Forge Storage cache for calculation results.

Architecture

The app renders a Jira issue panel and calls a Forge resolver to retrieve issue data through Jira APIs. It does not use Tempo APIs and does not calculate rates or costs.

Permissions

read:jira-work is used to read issue data. storage:app is used for app-scoped cache.

Support contact

Report security questions to support.jira@mederak.pl.
