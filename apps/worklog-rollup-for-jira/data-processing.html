Data processing

Data Processing

The app processes only the Jira data needed to calculate logged time rollups.

Purpose

Calculate logged time for the current Jira issue and descendant issues.

Data categories

Issue keys, summaries, issue types, parent relationships, issue counts, hierarchy depth and time tracking values.

Storage

Short-lived calculation cache may be stored in Forge Storage.
