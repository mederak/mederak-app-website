Subprocessors

This page lists core service providers involved in providing Project Overview & Status Hub for Jira.

Name Purpose Data involved
Atlassian Jira Cloud, Forge app hosting, Forge app storage, Forge permissions, Forge LLM capabilities, Marketplace installation and licensing. Jira data available to the app, generated app records, app configuration, user and site context.
Google Analytics Optional product analytics when configured for the production environment and public website measurement. Sanitized usage events and request metadata. The app is designed not to send Jira issue content, saved overview text, project names, user names, email addresses or links.

Subprocessor changes

Material changes to subprocessors will be reflected on this page. Customers can contact Mederak Apps Service Desk with questions.

Website analytics

Google measurement on this public marketing website is separate from the Forge app runtime and does not process customer Jira issue content from the installed app.
