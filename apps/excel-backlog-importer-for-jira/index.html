Why Forge matters

Hosted on Atlassian infrastructure, not on a separate vendor server.

Atlassian describes Forge as its serverless cloud app platform with compute and storage hosted on Atlassian infrastructure. Forge apps also run with tenant isolation and manifest-controlled external egress.

✓ Atlassian-hosted compute and storage options reduce the need for a separate vendor backend. ✓ The app is designed not to store full Excel files and not to send backlog data to the app creator's servers. ✓ Forge storage is scoped per app installation and can align with Atlassian data residency capabilities where eligible. ✓ External data egress is controlled through the Forge manifest. This app is designed without a remote product backend.

Why not just use Jira CSV import?

Jira's built-in CSV import is powerful, but it is usually admin-oriented. Excel Backlog Importer is designed for project teams who need a guided, repeatable import workflow inside an existing project.
